# SpeedClean
(--- LOGICIEL NON FONCTIONNEL ET EN COURS DE DEVELOPPEMENT ---)
--
## Introduction
Ce logiciel est un utilitaire développé en C# qui permet de supprimer les fichiers temporaires sous Windows 10/11.
## Fonctionnement
- Télécharger et lancer le fichier "SpeedClean.exe".
- Cliquer sur le bouton "Analyser" et vous verrez apparaitre la taille des fichiers temporaires.
- CLiquer sur "Nettoyer" pour supprimer définitivement l'ensemble des fichiers temporaires."
